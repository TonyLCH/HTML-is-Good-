# 7.站內超連結
#### 在為標籤設定id之後，超連結的目標指向那個id，就可以自動跳躍至頁面的那個位置
#### 對於網頁裡面含有很長的文檔時非常適合當作索引
#### &lt;標籤 id="標籤名稱" &gt;
#### &lt;a href="#標籤名稱"&gt;連結文字&lt;/a&gt;
